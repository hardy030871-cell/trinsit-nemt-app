# TRINSIT Rebuild v1

Clean rebuilt foundation for the TRINSIT web app.

## Includes
- PIN-only login
- Admin, dispatcher, driver, contractor driver roles
- Attendance with admin manual clock-in/clock-out override
- Trips with dispatcher board and strict driver trip progression
- Live map refresh with clocked-in user locations
- Expenses CRUD with admin delete
- Team chat
- User management
- Control panel foundation

## Run
npm install
npm start

Then open http://localhost:3000

## Default PINs
- Admin: 001900!
- Dispatcher: 222222!
- Driver: 333333!
- Contractor: 444444!
